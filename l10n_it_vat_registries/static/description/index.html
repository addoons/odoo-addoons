Aggiunta della reportistica dei registri IVA.

Registro IVA

Aggiunta la voce di menu "Registri IVA" al menu "Rendicontazione dell'app "Contabilità". Una volta cliccato sulla voce di menu, si apre un wizard per configurare la stampa dei registri IVA. Le sezioni più importanti sono:

  • Data: tramite un range di date (ad esempio un trimestre) o tramite data inizio e data fine è possibile selezionare lo spazio temporale d'interesse.
  • Registri: tramite una tabella è possibile selezionare quali registri considerare per stampare le voci
  • Altre opzioni: è possibile definire alcune caratteristiche del layout

Altre aggiunte

  • Tasse: checkbox per escludere una certa imposta dal registro IVA
  • Conti: campo relazionale per assegnare un registro iva
  • Registro IVA: aggiunta la voce di menu "Registri IVA" nel menu "Configurazione" dell'app "Contabilità". Cliccando si apre la vista ad elenco dei registri IVA configurati. Cliccando su una riga si apre la form dedicata al registro IVA selezionato contenente: Nome, Struttura, tabella dei registri legati al registro IVA
